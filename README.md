# Biscotti

## これはなに？

VRアプリケーションやスクリーンショットツールが書き出す画像フォルダを監視し、画像内のQRコードを自動で読み取って履歴化するWindows向けデスクトップアプリケーションです。

## 主な機能

- 指定フォルダの再帰的監視と新規画像の検知
- ファイル書き込み完了の自動判定
- QRコードのデコード
- 読み取り履歴の永続化
- ライト/ダークテーマ対応

## 基本的な利用方法

1. 配布された `biscotti.exe` を起動します。
2. 初回起動時にフォルダー選択ダイアログが表示されるため、画像が保存されるフォルダーを選択します。
3. 画面右上の「監視開始」を押すと、新しく作成された画像の監視を開始します。
4. QRコードを含む画像が見つかると、自動で読み取り履歴に追加されます。
5. 履歴を選択すると、画像プレビューと読み取り結果を確認できます。

## アンインストール方法

1. 本アプリケーションを実行中の場合、プロセスを終了します。
2. 配置した `biscotti.exe` を削除します。
3. 設定や読み取り履歴も削除する場合は、以下のフォルダーを削除します。

```text
%APPDATA%\Biscotti
```
